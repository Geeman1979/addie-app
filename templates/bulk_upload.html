Quick Tips
  • Only the title column is required — everything else is optional
  • Use completed_phases to mark which ADDIE phases were done (comma-separated: analysis,design,development,implementation,evaluation)
  • If evaluation is in completed_phases, the request is marked as Completed
  • Dates must be in YYYY-MM-DD format
  • After importing, go to each request to upload learning stats (CSV per intervention)
